# CareSync

### Quickstart

To set up: Follow the instructions on https://docs.flutter.dev/get-started/quick until "Test drive Flutter". 

Clone this repo, open it in the terminal. 

To run the application locally on chrome:
```
flutter run -d chrome
```
If you have issues running, run 

```
flutter clean
```

To get dependencies, run 

```
flutter pub get
```

Once it is running locally in chrome, right click and click inspect. Then, click the button to the left of "elements" to choose the dimensions (ex iphone 14 dimensions).
